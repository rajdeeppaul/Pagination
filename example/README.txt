Instruction to run example code(s):

 1. Download and import `world database` from <a href="https://dev.mysql.com/doc/index-other.html">https://dev.mysql.com/doc/index-other.html</a>

 3. Run `index.php` from your server's root directory like this, http://localhost/example/index.php
